# 🚀 Network Health Monitor Dashboard

A real-time network monitoring system that tracks host availability, latency, and uptime with live visualization.

## 🔥 Features
- Real-time network monitoring
- Live dashboard using Flask
- Latency tracking (ms)
- Uptime percentage calculation
- Email alerts for downtime
- Interactive graphs using Chart.js

## 🛠 Tech Stack
- Python
- Flask
- JavaScript (Chart.js)
- HTML/CSS

## 🚀 Run Project
```bash
pip install flask requests
python app.py
